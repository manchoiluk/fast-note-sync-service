## v3.1.1
> *2026/05/25*

### 🚀 優化

## v3.0.6
> *2026/05/23*

### ✨ 新增

- **MCP**: 新增 `file_write` 工具，支援第三方用戶端透過 Base64 編碼直接寫入附件或檔案並即時同步至各端（請避免上傳超大檔案，以防記憶體不足發生 OOM）。
- **網路**: 新增自訂 `header` 回應標頭設定，支援在設定檔中為所有 HTTP 服務和 WebSocket 握手階段自訂回應標頭（詳見 `config.yaml` 中的 `server.custom-response-headers` 選項）。

### 🚀 優化

- **WebGUI**: 優化筆記列表移動互動。支援透過滑鼠拖曳移動筆記或附件到根目錄或其他子資料夾。
- **WebGUI**: 優化 Obsidian 屬性轉譯。筆記編輯頁和分享筆記頁現在可以正確顯示 Obsidian 筆記的屬性（Properties）。
- **WebGUI**: 優化分享筆記入口。在筆記編輯頁增加一鍵「分享筆記」的按鈕入口。

### 🛠️ 修復

- **自啟動**: 修復 macOS 自啟動服務因權限不足導致無法正常執行，以及快捷安裝指令碼執行報錯的問題。
- **配置**: 修復設定檔中顯式將預設開啟的布林配置（如 `webgui-login-token-bind-ip`, 雲端儲存啟用開關及 WebSocket 並行/壓縮選項）設為 `false` 後依然被預設值覆蓋的問題。
- **WebGUI**: 修復 WebGUI 介面在載入圖片或附件資源時，由於瀏覽器請求未攜帶特定用戶端 Header 導致檔案被攔截報錯的問題。

## v3.0.0
> *2026/05/14*

> [!WARNING]
> **注意事項**：本次更新引入了全新的令牌架構，會導致舊版所有授權失效。升級後您需要對所有用戶端設備進行重新授權。

### 🔐 核心更新：全新有狀態令牌權限系統

本項目重構了底層的安全架構，從單一無狀態令牌升級為 **有狀態多維度（協定、用戶端、功能）** 校驗權杖：
- **多維度校驗**：
  - **協定控制**：支援對 `REST API`、`WebSocket`、`MCP` 等不同存取協定及場景進行獨立授權與限制。
  - **用戶端限制**：支援透過萬用字元綁定特定裝置（如 `obsidian*`），實現「一機一令」。
  - **內容控制**：細化至筆記讀寫 (`note_r/w`)、附件管理 (`file_r/w`)、系統設定 (`config_r/w`) 等原子級權限。
- **世代校驗與權杖輪轉 (Rotation)**：
  - 引入 `Nonce` 世代機制。權杖在存在洩漏風險時可進行輪轉操作，舊世代權杖將立即失效，有效防範權杖被盜用的風險。
- **動態環境綁定**：
  - 支援對權杖進行 **IP 地址** 和 **User-Agent** 的靜態或萬用字元綁定，用於在非安全環境下限制權杖使用行為。
- **權杖存取統計**：
  - 伺服器端將會記錄所有權杖的存取行為，支援即時監控各權杖的呼叫協定、來源 IP 及用戶端版本。

### ✨ 新增

- **備份**: 新增備份任務密碼加密設定。全量與增量備份任務新增靈活的備份包加密策略，支援「不設密碼、手動固定密碼、系統隨機密碼」三種設定模式，顯著提升備份資料的儲存安全性。

### 🛠️ 修復

- **雲儲存**: 升級阿里雲 OSS v2 SDK 驅動，修復儲存設定連線測試失敗及區域識別問題。

### 🚀 優化

- **效能**: 優化檔案處理效能。調整雜湊計算閾值為 10MB，透過延遲建立實體資料夾大幅減少檔案系統 I/O 開銷。
- **協定**: 優化協定標準化。所有 WebSocket 返回訊息強制包含 `vault` 欄位，確保多庫同步狀態的一致性。
- **安全**: 安全加固。備份歷史記錄中的敏感密碼欄位預設開啟遮罩處理，支援手動解密查看。
